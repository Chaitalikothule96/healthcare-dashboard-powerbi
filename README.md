📊 Healthcare Dashboard (Power BI)

🔹 Problem Statement
Analyze healthcare data to monitor patient trends, treatment costs, and hospital performance.

🔹 Tools Used
Power BI, DAX, Data Modeling

🔹 Dataset
Healthcare dataset including patient records, treatment details, and costs

🔹 Key Insights

Identified patient trends across departments
Analyzed treatment costs and revenue patterns
Found high-performing and low-performing departments
Improved visibility of hospital performance using KPIs

🔹 Dashboard Features

Interactive filters and slicers
KPI metrics (Total Patients, Treatment Cost, Revenue)
Department-wise and time-based analysis
